def detectar_atipicos(df_train, df_val, cols_diana):
    for col in cols_diana:

        # Calculamos IQR y lmites
        Q1 = df_train[col].quantile(0.25)
        Q3 = df_train[col].quantile(0.75)
        IQR = Q3 - Q1
        lower = Q1 - 6 * IQR
        upper = Q3 + 6 * IQR
        
        # Declaramos condicin de atpico y registramos sus posiciones
        cond_iqr = (df_train[col] < lower) | (df_train[col] > upper)    #OLLO! Nico recomendara ver os casos 1 a 1 
        cond_neg = df_train[col] < 0
        cond_atipico = cond_iqr | cond_neg

        cond_iqr_val = (df_val[col] < lower) | (df_val[col] > upper)    #OLLO! Nico recomendara ver os casos 1 a 1 
        cond_neg_val = df_val[col] < 0
        cond_atipico_val = cond_iqr_val | cond_neg_val

        df_train.loc[cond_atipico, col] = pd.NA
        df_val.loc[cond_atipico_val, col] = pd.NA


    return df_train, df_val

1.3.1 - Tratamiento de datos atpicos

Tal e como se comentou arriba, agora que tdolos atpicos convertronse a null, podemos tratalos conxuntamente cos valores null orixinais. Decidimos que o lmite de valores permitidos fose 3, dicir:

  • No conxunto test, eliminaranse as filas con 4 ou mis valores atpicos
  • No resto de casos (filas con menos de 4 valores null no conxunto test, e todos os null do conxunto validacin) aplicarase unha imputacin de valores empregando a mediana do conxunto test.

A decisin de tomar como cantidade lmite 3 valores por fila veu dada porque a maiora das filas presentaban entre 0 e 2 valores atpicos (pdese comprobar no resultado mostrado 2 celdas mis arriba). Comprobando que a distribucin de valores de atpicos se reflexaba na eleccin do valor m do IQR, decidimos que 3 era un valor adecuado.

def eliminar_o_imputacion(df_train, df_val, cols_diana, max_atipicos=3, umbral_col=20,  target='calidad'):
    # Eliminar filas con 4 o ms valores atpicos
    # Contamos el nmero de valores atpicos por fila (NA) y filtramos 
    df_train = df_train[df_train.isnull().sum(axis=1) < max_atipicos]

    # Analisis de valores faltantes por columna
    for col in cols_diana:
        # Contamos el numero de valores faltantes (NA) en la columna(
        num_faltantes = df_train[col].isna().sum()
        if num_faltantes/len(df_train) > umbral_col:
            df_train = df_train.drop(columns=[col])
            df_val = df_val.drop(columns=[col])

    # Imputacion de valores: mediana para cada nulo de cada columna
    for col in cols_diana:
        med = df_train[col].median()
        df_train.loc[:, col] = df_train[col].fillna(med)
        df_val.loc[:, col] = df_val[col].fillna(med)

    return df_train, df_val

Na seguinte funcin recollemos a validacin cruzada anidada. Nela dividiremos o conxunto en 6 particins, sobre as que realizaremos:

  • Una validacin exterior consistente en 3 iteracins, no que un grupo de 2 particins actuar como conxunto de test externo e as demis como conxunto de adestramento externo
  • En cada unha desas iteracins, realizarase unha validacin cruzada interna sobre as 4 particins de adestramento externas.
  • Esta validacin interna farase tantas veces como combinacins de hiperparmetros se definisen, ata obter a combinacin cun maior rendemento.
  • Empregando a combinacin de hiperparmetros ptima, adestrarase o modelo no conxunto de adestramento externo e validarase no conxunto de proba externo.

PROCESO:

  1. Comezamos realizando un preprocesado inicial sobre o dataset obxectivo, eliminando duplicados e convertndoo nunha clasificacin binaria. A continuacin mezclamos o dataset para unha maior aleatoriedade.

  2. Procdese a dividilos datos nas 6 particins que se empregarn para a validacim cruazda.

  3. Dentro de cada iteracin externa:

  • 3.1. Constrese un diccionario para cada iteracin externa: almacnase nun array de puntuacin para cada modelo evaluado na validacin cruzada interna. Conter:

    • Nome
    • Modelo
    • Parmetros: Combinacin de hiperparmetros empregada
    • Accuracy: Porcentaxe total de acertos na predicin
    • Precision: Porcentaxe de predicins positivas realmente positivas
    • Recall: Porcentaxe dos positivos predecidos correctamente
    • F1: Medida que representa o equilibrio entre a precisin e o recall O que tea mellor media ser o usado para adestrar co conxunto completo na iteracin externa e evaliar na sa validacin externa.

  • 3.2. Repartimos as 2 particins de validacin e as 4 de adestramento.

  • 3.3. Dentro de cada iteracin interna:

    • Preprocesamos os conxuntos e separamos caractersticas e etiquetas para adestramento e validacin.
    • Para particin efectase un bucle que recorre as distintas combinacins de hiperparmetros definidas para todos os modelos
      • Entrnase o modelo con cada combinacin e avalase en validacin interna

  • 3.4. Ao finalizala validacin interna, calclase a media dos resultados

  1. Finalizadas as iteracins externas, eliminamos duplicados de presentes nos mellores resultados obtidos na validacin interna para que non haia configuracins repetidas na avaliacin externa

  2. A continuacin evalanse os modelos escollidos entrenndoos e validndoos contra os tres conxuntos exteriores.

  3. Ao finalizar as iteracins calclanse as medias para as mtricas producidas e obtense o rendemento final de cada configuracin.

  4. Por ltimo, fltranse as mellores configuracins para cada modelo seguindo as medias anteriores e devlvense.

def ejecucion_prediccion(file_train, file_test, configuracion_modelo):
    df_train = pd.read_csv(file_train)
    df_test = pd.read_csv(file_test)
    
    # preprocesado inicial
    df_train = eliminacion_duplicados(df_train)
    df_train = binarizar_calidad(df_train)
    df_test = eliminacion_duplicados(df_test)
    df_test = binarizar_calidad(df_test)
    
    # Preprocesar
    df_train_proc, df_test_proc = preprocesado(df_train, df_test, target='calidad')
    # Separar caractersticas y etiquetas
    X_train = df_train_proc.drop(columns=['calidad'])
    y_train = df_train_proc['calidad']
    X_test = df_test_proc.drop(columns=['calidad'])
    y_test = df_test_proc['calidad']

    # Entrenar el modelo con la configuracin dada
    modelo = configuracion_modelo["model"](**configuracion_modelo["params"])
    modelo = entrenar_o_cargar(modelo, X_train, y_train, f"results/modelo_final_{configuracion_modelo['name']}_params{tuple(configuracion_modelo['params'].values())}.pkl")

    # Hacer predicciones
    y_pred = modelo.predict(X_test)
    # Calcular mtricas
    resultados = []
    resultados.append({"accuracy": accuracy_score(y_test, y_pred)})
    resultados.append({"precision": precision_score(y_test, y_pred)})
    resultados.append({"recall": recall_score(y_test, y_pred)})
    resultados.append({"f1_score": f1_score(y_test, y_pred)})
    return resultados
